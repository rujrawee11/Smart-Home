บทคัดย่อ



เคยลืมปิดไฟไหม? เคยออกไปข้างนอกลืมเก็บผ้าแล้วฝนตกไหม? คงเป็นปัญหาที่หลายๆคนเคยเจอ
คณะผู้จัดทำโครงงานจึงได้ทำแบบจำลองวงจรภายในบ้าน โดยให้สามารถควบคุมการเปิด-ปิดไฟได้โดยใช้แอปพลิเคชันBlynk แล้วยังสามารถสั่งงานด้วยเสียงได้ (google assistant)
เปิด-ปิดไฟแบบอัตโนมัติ เมื่อมีแสงหลอดไฟจะดับ ไม่มีแสงหลอดไฟจะติด (เช่น ไฟหน้าบ้าน, ไฟของรั้ว)
มีการส่งแจ้งเตือนไปเมื่อฝนตก(เช่น ออกไปข้างนอกแต่ฝนตก คนในบ้านไม่รู้ว่าตากผ้าอยู่สามารถบอกให้คนในบ้านมาเก็บผ้าให้ได้)
และมีการส่งสัญญาญเตือนเมื่อเจอสิ่งกีดขวาง (เช่น มีคนมาบุกรุกอาจจะเป็นเพื่อนหรือโจร) โดยการต่อแบบจำลองนี้
คณะผู้จัดทำได้ใช้ภาษา C ในการเขียนโปรแกรม ร่วมกับการต่อบอร์ดโดยใช้ NodeMCU ในการควบคุม เป็นการปฏิบัติการแบบ IoT หรือ Internet of Things
ซึ่งในปัจจุบัน IoT ได้เข้ามามีบทบาทอย่างมากในชีวิตประจำวัน
คณะผู้จัดทำโครงงานจึงได้นำความรู้มาประยุกต์ใช้ให้เข้ากับการดำเนินชีวิตในแต่ละวันเพื่อเพิ่มความสะดวกสบายให้กับผู้ใช้งานมากขึ้น ทางคณะผู้จัดทำหวังเป็นอย่างยิ่งว่า
โครงงานแบบจำลองวงจรภายในบ้าน จะสามารถสร้างประโยชน์และเป็นแนวทางในการพัฒนาเทคโนโลยีที่เกี่ยวข้องเกี่ยว IoT ต่อไปในอนาคตเป็นอย่างมาก

จุดประสงค์



1. เพื่อให้สะดวกต่อการเปิดปิดไฟในบ้านมากขึ้นและช่วยลดค่าใช้จ่าย (ลืมปิดไฟน้อยลง)
2. เพื่อให้ลดการวิตกกังวล เวลาที่ออกไปข้างนอกแล้วตากผ้าไว้ ให้แบบจำลองนี้ส่งสัญญาณไปแจ้งเตือน
3. เพื่อให้มีการระวังตัวเองมากขึ้นเมื่อมีสัญญาณเตือนว่ามีคนมาบุกรุก ให้ดูกล้องวงจรปิด หรือไปแอบดูว่าใครมา



อุปกรณ์

  • บอร์ดทดลองวงจร (Photoboard)
  • Node MCU V2 ESP8266
  • LED
  • ตัวต้านทาง 220 โอห์ม
  • Jumper M2M, Jumper M2F
  • LDR photoresistor sensor วัดความสว่างความเข้มแสง
  • Raindrop sensor วัดปริมาณน้ำฝน
  • IR infrared obstacle avoidance sensor ตรวจจับวัตถุสิ่งกีดขวาง

    • หลักการทำงานของแบบจำลองการต่อวงจรภายในบ้าน

    ตามแบบจำลองวงจรภายในบ้าน โดยให้ LED จำลองเป็นหลอดไฟภายในบ้าน 2 หลอดและมีไฟเตือนด้วย 2 หลอด

    1. การเปิด-ปิดไฟ โดยการกดปุ่มในแอปพลิเคชัน Blynk จะต้องไปทำการสร้างปุ่มใน
    Blynk และทำการเชื่อมกับ code ก่อนถึงจะทำการควบคุมการเปิดปิดได้ หรือสามารถใช้การสั่งงานด้วยเสียง(google assistant)
    2. การเปิด-ปิดไฟแบบอัตโนมัติด้วยการวัดปริมาณตามความเข้มแสง ถ้ามีแสง จะทำให้หลอดไฟดับ
    ถ้าไม่มีแสงจะทำให้หลอดไฟจะติด
    3. เมื่อมีน้ำที่ raindrop จะมีไฟแจ้งเตือนที่ LED ว่าฝนตกแล้วจะส่งแจ้งเตือนไปทางไลน์ว่า ฝนตกแล้ว
    4. เมื่อมีสิ่งกีดขวางที่หน้า IR จะทำการแจ้งเตือนที่ LED และส่งแจ้งเตือนไปทางไลน์
    จากนี้และนานถึงเมื่อไหร่ถ้าไม่ยอมตัดขาดเธอไปต้องเจ็บวนไปอยู่อย่างนี้จะเลิกเอาใจไปรักคนที่ไม่รักเราจะเลิกเอาใจไปทิ้ง

    Source Code



    เพราะมีความรักฉันเลยเป็นคนที่อ่อนแอพบกับความพ่ายแพ้ที่วนเวียนซ้ำอย่างทุกข์ทนทุกครั้งยังคอยตอกย้ำเจ็บแล้วไม่จำไม่ใช่คน
    ถ้าต้องการหลุดจากวังวนต้องไม่ยอมทนอยู่อย่างนี้จะเลิกเอาใจไปรักคนที่ไม่รักเราจะเลิกเอาใจไปทิ้ง
    ให้เจ็บให้ปวดร้าวเพราะทำดีเท่าไหร่ก็ได้เพียงความว่างเปล่าถ้าหากทุ่มเทให้คนที่ไม่มีหัวใจจะต้องเป็นคนเข้มแข็ง
    ต่อจากนี้ไปจะเลิกเป็นคนอ่อนไหวให้คนที่ทำร้ายกันแต่คิดจะลืมเท่าไหร่กลับไม่เคยลืมสักวันมันไม่เข้าใจทำไม
    สุดท้ายก็ยังไปรักเธอเพราะว่าเธอคนนี้ที่มีให้เพียงความผิดหวังเสียน้ำตากี่ครั้งหัวใจก็ยังให้อภัยทุกครั้งมันเป็นคำถาม
    จากนี้และนานถึงเมื่อไหร่ถ้าไม่ยอมตัดขาดเธอไปต้องเจ็บวนไปอยู่อย่างนี้จะเลิกเอาใจไปรักคนที่ไม่รักเราจะเลิกเอาใจไปทิ้ง
