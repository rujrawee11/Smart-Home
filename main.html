บทคัดย่อ



เคยลืมปิดไฟไหม? เคยออกไปข้างนอกลืมเก็บผ้าแล้วฝนตกไหม? คงเป็นปัญหาที่หลายๆคนเคยเจอ คณะผู้จัดทำโครงงานจึงได้ทำแบบจำลองวงจร
ภายในบ้านโดยให้สามารถควบคุมการเปิด-ปิดไฟได้โดยใช้แอปพลิเคชันBlynk แล้วยังสามารถสั่งงานด้วยเสียงได้ (google assistant)เปิด-ปิดไฟ
แบบอัตโนมัติเมื่อมีแสงหลอดไฟจะดับ ไม่มีแสงหลอดไฟจะติด (เช่น ไฟหน้าบ้าน, ไฟของรั้ว) มีการส่งแจ้งเตือนไปเมื่อฝนตก (เช่น ออกไปข้างนอกแต่
ฝนตก คนในบ้านไม่รู้ว่าตากผ้าอยู่สามารถบอกให้คนในบ้านมาเก็บผ้าให้ได้) และมีการส่งสัญญาญเตือนเมื่อเจอสิ่งกีดขวาง (เช่น มีคนมาบุกรุกอาจ
จะเป็นเพื่อนหรือโจร) โดยการต่อแบบจำลองนี้ คณะผู้จัดทำได้ใช้ภาษา C ในการเขียนโปรแกรม ร่วมกับการต่อบอร์ดโดยใช้ NodeMCU ในการควบ
คุมเป็นการปฏิบัติการแบบ IoT หรือ Internet of Things ซึ่งในปัจจุบัน IoT ได้เข้ามามีบทบาทอย่างมากในชีวิตประจำวัน คณะผู้จัดทำโครงงานจึง
ได้นำความรู้มาประยุกต์ใช้ให้เข้ากับการดำเนินชีวิตในแต่ละวันเพื่อเพิ่มความสะดวกสบายให้กับผู้ใช้งานมากขึ้น ทางคณะผู้จัดทำหวังเป็นอย่างยิ่ง
ว่าโครงงานแบบจำลองวงจรภายในบ้าน จะสามารถสร้างประโยชน์และเป็นแนวทางในการพัฒนาเทคโนโลยีที่เกี่ยวข้องเกี่ยว IoT ต่อไปในอนาคต
เป็นอย่างมาก

จุดประสงค์



1. เพื่อให้สะดวกต่อการเปิดปิดไฟในบ้านมากขึ้นและช่วยลดค่าใช้จ่าย (ลืมปิดไฟน้อยลง)
2. เพื่อให้ลดการวิตกกังวล เวลาที่ออกไปข้างนอกแล้วตากผ้าไว้ ให้แบบจำลองนี้ส่งสัญญาณไปแจ้งเตือน
3. เพื่อให้มีการระวังตัวเองมากขึ้นเมื่อมีสัญญาณเตือนว่ามีคนมาบุกรุก ให้ดูกล้องวงจรปิด หรือไปแอบดูว่าใครมา

       ตามแบบจำลองวงจรภายในบ้าน โดยให้ LED จำลองเป็นหลอดไฟภายในบ้าน 2 หลอดและมีไฟเตือนด้วย 2 หลอด

        1. การเปิด-ปิดไฟ โดยการกดปุ่มในแอปพลิเคชัน Blynk จะต้องไปทำการสร้างปุ่มใน Blynk
        และทำการเชื่อมกับ code ก่อนถึงจะทำการควบคุมการเปิดปิดได้ หรือสามารถใช้การสั่งงานด้วยเสียง(google assistant)

        2. การเปิด-ปิดไฟแบบอัตโนมัติด้วยการวัดปริมาณตามความเข้มแสง ถ้ามีแสง จะทำให้หลอดไฟดับ ถ้าไม่มีแสงจะทำให้หลอดไฟจะติด

        3. เมื่อมีน้ำที่ raindrop จะมีไฟแจ้งเตือนที่ LED ว่าฝนตกแล้วจะส่งแจ้งเตือนไปทางไลน์ว่า ฝนตกแล้ว

        4. เมื่อมีสิ่งกีดขวางที่หน้า IR จะทำการแจ้งเตือนที่ LED และส่งแจ้งเตือนไปทางไลน์

    void setup() {
      Serial.begin(9600);
      Serial.println(LINE.getVersion());
      WiFi.begin(SSID, PASSWORD);

      Serial.printf("\nWiFi connected\nIP : ");
      Serial.println(WiFi.localIP());
      // กำหนด Line Token
      LINE.setToken(LINE_TOKEN);
      LINE.notify("myarduino.net");
      Blynk.begin(auth, ssid, pass); //Token, ชื่อ Wi-Fi, รหัสผ่าน

      pinMode(LED_PIN_4, OUTPUT); //แสดงค่าทาง Led Alarm sensor ตรวจจับสิ่งกีดขวาง

    วิธีการใช้งาน



    1.  ดาวน์โหลดโปรแกรม Arduino เมื่อเรียบร้อย ให้ติดตั้งแพลตฟอร์ม ESP8266
             - เลือกแถบ File -> Preferences
             - ไปที่ช่อง Additional Boards Manager URLs: ป้อน URL นี้ http://arduino.esp8266.com/stable/package_esp8266com_index.json แล้ว click OK
             - เลือกแถบ Tools -> Board: แล้วเลือกที่ Boards Manager...
             - ค้นหา esp8266 by ESP8266 Community จากนั้นกด Install
             - เมื่อติดตั้งเสร็จให้เลือก Board : NodeMCU 1.0 (ESP-12E Module)
    2.  หลังจากนั้นให้ติดตั้ง Library Blynk ตามข้างบนให้ครบ ดังนี้
             - เลือกแถบ Sketch --> Include Library --> Add .ZIP Library...
    3.  ดาวน์โหลดแอปพลิเคชัน Blynk ในโทรศัพท์มือถือ
             - เมื่อติดตั้งเสร็จให้เข้าแอปพลิเคชัน Blynk แล้วทำการ Log In
              -จากนั้นเลือก New Project และทำการตั้งค่าตามรูปภาพ 
             - แล้วกด Create จากนั้นจะได้รับ Auth Token ตาม E-mail ที่ใช้เชื่อมต่อกับแอพลิเคชัน เพื่อนำไปใส่ในโค้ด
             - ทำการเพิ่มปุ่มกด Button และตั้งค่า PIN เป็น Digital และตั้งค่าตามภาพ
             - เมื่อทำการตั้งค่าเสร็จให้เปิดไฟล์ของเราในโปรแกรม Arduino
             - จากนั้นคลิก Tools แล้วเลือก Port ของสาย และทำการอัปโหลดโค้ดผ่านสาย Micro USB เข้า NodeMCU ของแบบจำลองวงจรภายในบ้าน
             - เมื่ออัปโหลดเสร็จ จะสามารถควบคุมการเปิดปิดไฟได้
